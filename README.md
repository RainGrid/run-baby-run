# Run Baby Run

Браузерный endless runner в стиле Chrome Dino: младенец перепрыгивает препятствия. Прыжок — пробел / стрелка вверх / тап.

## Запуск

Открыть `index.html` в браузере или раздать папку статически.

## GitHub Pages

В настройках репозитория: **Settings → Pages → Source**: Deploy from a branch → **Branch**: `main`, **Folder**: `/ (root)` → Save. Сайт будет доступен по `https://raingrid.github.io/run-baby-run/`.

Файл `.nojekyll` в корне отключает Jekyll, чтобы все файлы отдавались как есть.
